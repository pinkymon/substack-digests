## Claude Code for Non-Coders — Prompts Don't Enforce Rules. Hooks Do.
[🔗 連結](https://claudecodefornoncoders.substack.com/p/prompts-dont-enforce-rules-hooks)

- Prompt 是「建議」，PreToolUse Hook 是「物理屏障」——Claude 會推理繞過指令，但無法繞過程式化攔截。

## Claude Cowork — Why vague tasks turn Claude Cowork into a token-burning machine
[🔗 連結](https://www.coworkoperator.com/p/why-claude-cowork-feels-expensive)

- Cowork 不是聊天，而是讓 Claude 讀寫你電腦上的檔案、拆解子任務、協調多個子代理——因此 token 消耗結構完全不同。
- 感覺貴的真正原因：多數人把它當「長時間通用助手」而非「範圍明確的工作面」來使用。
- Anthropic 在尖峰時段收緊了五小時會話限制，加上大量新用戶湧入，體感成本進一步放大。
